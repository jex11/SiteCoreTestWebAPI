# SiteCoreTestWebAPI

1. Download SiteCoreTestWebAPI ZIP file from github
2. Unzip and paste to local PC
3. Open project folder and right click on SiteCoreTestWebAPI.csproj file
4. Open with your Visual Studio 2017
5. Save solution after initialize project
6. Please make sure the SiteCoreTestWebAPI project is running
7. Please ensure this project running before SiteCoreTestApplication.
8. Compile and run the application on browser
9. Done
